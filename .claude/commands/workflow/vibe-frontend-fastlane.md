# /vibe-frontend-fastlane

Objetivo: construir frontend de alta qualidade rapidamente, guiado por referências reais.

Argumentos: `$ARGUMENTS`

## Etapas
1. Selecionar até 3 referências visuais de alta aderência ao objetivo.
2. Para cada referência, baixar/normalizar HTML com Website-Downloader.
3. Rodar extração de padrões (`/vibe-extract`).
4. Consolidar um mini design system do projeto atual:
- tipografia
- cores/superfícies
- componentes
- motion
- estados de UI
5. Implementar em ordem:
- layout base
- hero + CTA
- seção principal de valor
- grid/list principal
- formulário/ação final
6. Validar qualidade:
- contraste
- teclado
- reduced-motion
- mobile

## Critérios de conclusão
- Tela com identidade clara e coesa.
- Reuso de padrões em vez de CSS improvisado.
- Tempo de construção reduzido em ciclos seguintes.
