# Frontend Vibe Pack

Pacote de referência para acelerar construção de frontend com qualidade visual alta.

## Conteúdo
- `VIBE_FRONTEND_SYSTEM.md`: sistema operacional de design e execução.
- `WEBSITE_DOWNLOADER_WORKFLOW.md`: fluxo com Website-Downloader.
- `generated/vibe-extraction-report.md`: extração automática das referências HTML.

## Uso rápido
1. Rode o Website-Downloader para capturar um site de referência.
2. Aplique o prompt de extração para gerar `design-system.html` fiel ao original.
3. Replique padrões com o playbook, sem inventar estilos fora da referência.

## Princípio
Velocidade vem de **consistência + biblioteca de padrões + fluxo repetível**.
